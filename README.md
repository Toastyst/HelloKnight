# HelloKnight

A 2D action-adventure game built in Godot where you play as a knight battling enemies in a dark fantasy world. Features stamina and health mechanics, collectibles, and atmospheric music.

## How to Play

- Clone the repository
- Open `project.godot` in Godot 4.x
- Run the main scene: `scenes/main scenes/game.tscn`

## Controls

- Movement: WASD or Arrow Keys
- Jump: Space
- Roll: Shift (while on ground, costs 20 stamina)
- Block: Hold Shift (reduces damage by half, drains 15 stamina)
- Light Attack: Left Mouse Button (costs 15 stamina)
- Heavy Attack: Right Mouse Button (costs 25 stamina)

## Features

- Health and Stamina systems with regeneration
- Combat with light/heavy attacks and blocking
- Rolling for evasion
- Collect coins and items
- Atmospheric music and sound effects
- Enemy AI and combat encounters
